# kaliningrad-rent-prediction

Проект представляет собой сбор информации про посуточную аренду в Калининграде с сайта Avito и последующий анализ этих данных

Папка Smirnov представляет собой работу Смирнов Антона.
---
- В программе parser_avito/avito_parser.py реализована сбор данных объявлений (поверхностный).
- В программе parser_avito/avito_parser_detailed.py реализована сбор данных уже более детальный (проходится по каждый странице и собирает всю информацию)
- В программе analysis/Smirnov.ipynb реализована очистка, статистика, немного графиков и предсказание различных моделей на уже собранных данных.

Папка Golubtsov представляет собой работу Голубцова Никиты
---
- Основная работа - анализ всего (осноная)
