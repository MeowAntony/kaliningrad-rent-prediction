# Прогнозирование стоимости посуточной аренды в Калининграде

Проект по сбору, анализу и прогнозированию стоимости посуточной аренды жилья в Калининграде на основе данных с платформы Avito. Включает парсинг объявлений, статистический анализ и построение моделей машинного обучения для предсказания цен.

## Краткое описание

Данный проект представляет собой комплексное исследование рынка посуточной аренды в Калининграде. Система автоматически собирает данные объявлений с Avito, выполняет их очистку и анализ, а затем строит прогнозные модели для оценки стоимости аренды на основе различных характеристик жилья.

**Основные возможности:**
- Автоматический парсинг объявлений с Avito (поверхностный и детальный)
- Очистка и предобработка собранных данных
- Статистический анализ и визуализация рынка аренды
- Построение и сравнение моделей машинного обучения для прогнозирования цен
- Анализ факторов, влияющих на стоимость аренды

## Инструкция по установке

### Требования

- Python 3.8 или выше
- pip
- Git

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/MeowAntony/kaliningrad-rent-prediction.git
cd kaliningrad-rent-prediction
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Инструкция по использованию

### Сбор данных

**Поверхностный парсинг:**
```bash
cd Smirnov/parser_avito
python avito_parser.py
```

**Детальный парсинг (проходит по каждому объявлению):**
```bash
cd Smirnov/parser_avito
python avito_parser_detailed.py
```

### Анализ данных

Проект содержит два основных Jupyter-ноутбука для анализа:

**1. Анализ Смирнова (Smirnov/analysis/Smirnov.ipynb)**

Этот ноутбук выполняет полный цикл обработки и анализа данных:

- **Загрузка данных**: Чтение данных из файла `data/avito_details.json`
- **Очистка данных**: 
  - Удаление дубликатов по URL
  - Преобразование типов данных (цена, площадь, количество комнат)
  - Обработка пропусков в данных
  - Удаление выбросов методом Z-score
- **Исследовательский анализ**:
  - Описательная статистика для цены и площади
  - Расчет корреляции между переменными
  - Анализ по районам города
  - Визуализация распределений
- **Моделирование**:
  - Построение моделей машинного обучения (Linear Regression, Random Forest, XGBoost)
  - Оценка качества моделей (MAE, RMSE, R²)
  - Сравнение результатов
  - Анализ важности признаков

**2. Анализ Голубцова (Golubtsov/Голубцов.ipynb)**

Основной ноутбук для комплексного анализа данных рынка аренды, включающий детальное исследование характеристик объявлений и рыночных тенденций.

### Запуск ноутбуков

Откройте Jupyter Notebook для интерактивного анализа:
```bash
# Анализ Смирнова
jupyter notebook Smirnov/analysis/Smirnov.ipynb

# Анализ Голубцова
jupyter notebook Golubtsov/Голубцов.ipynb
```
